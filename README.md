# Animal-Image-Classification-CNN
A Convolutional Neural Network (CNN) model built using TensorFlow/Keras to classify five animal species (cat, cow, deep, dog, lion) with training, validation, evaluation metrics, and visualization of predictions.

## Dataset
Kaggle Animal Image Classification (5 Species)
https://www.kaggle.com/datasets/miadul/animal-image-classification-5-species?resource=download

## Model
- 3 Convolutional Layers
- MaxPooling Layers
- Fully Connected Layer
- Softmax Output Layer

## Results
- Validation Accuracy: 100%
- Perfect confusion matrix (all predictions correct)

- Jupyter Notebook
